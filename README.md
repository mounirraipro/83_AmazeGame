# Amaze Game

- **Domain:** amaze-game.com

- **Slug:** amaze-game
- **Genre:** Casual
- **Iframe:** https://amaze-game.com

## Quick start

```bash
npm install
npm run dev
```

## Content

All page content is driven by `content/game.json`, auto-generated by Claude API.
To regenerate:

```bash
ANTHROPIC_API_KEY=sk-... node ../generate-content.js \
  --name   "Amaze Game" \
  --domain "amaze-game.com" \
  --slug   "amaze-game" \
  --genre  "Casual" \
  --iframe "https://amaze-game.com" \
  --out    ./content/game.json
```

## What to customise manually

- Images in `public/` — replace with your actual game assets
- `content/game.json` — tweak any generated copy
